## Descripción General

Este repositorio contiene el portafolio de proyectos desarrollados durante el semestre en la materia de Administración y Organización de Datos.

Los proyectos muestran la aplicación de diferentes organizaciones de archivos en situaciones reales, utilizando formatos como JSON, CSV y TXT para almacenar, procesar y consultar información.

La competencia principal demostrada es:

> Evaluar diferentes organizaciones de archivos aplicándolas a situaciones reales.
