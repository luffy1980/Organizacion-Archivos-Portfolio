# Proyecto 3: Actividad Integradora C3

## Descripción

Este proyecto corresponde a la Actividad Integradora de Evaluación C3. Su propósito es integrar los conocimientos desarrollados durante el semestre sobre administración y organización de archivos, aplicándolos en un sistema funcional con manejo de datos mediante archivos.

## Objetivo

Desarrollar un sistema integrador que permita almacenar, procesar y consultar información utilizando diferentes formatos de archivo y una estructura organizada de software.

## Tecnologías Utilizadas

- Python
- PHP
- HTML
- Archivos locales

## Formatos de Archivo

| Formato | Uso |
|---|---|
| JSON | Manejo de información estructurada |
| CSV | Almacenamiento tabular de registros |
| TXT | Registro secuencial de eventos o datos simples |
