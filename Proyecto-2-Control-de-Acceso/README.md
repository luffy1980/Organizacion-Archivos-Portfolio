# Proyecto 2: Control de Acceso

## Descripción

Este proyecto corresponde a la Actividad de Evaluación – ABPj: Control de Acceso. El sistema tiene como propósito administrar registros relacionados con el acceso de usuarios, aplicando conceptos de organización de archivos para almacenar, consultar y validar información.

## Objetivo

Desarrollar un sistema de control de acceso que permita registrar y consultar información utilizando archivos como medio de almacenamiento.

## Tecnologías Utilizadas

- Python
- PHP
- HTML
- Archivos locales

## Formatos de Archivo

| Formato | Uso |
|---|---|
| JSON | Almacenamiento estructurado de usuarios o configuraciones |
| CSV | Registro tabular de accesos |
| TXT | Bitácora o historial de eventos |
